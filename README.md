# MoodleFusion

A separate 3-page responsive portfolio website created without modifying the original `24BCE2353` task files.

## Pages

- `index.html` - project overview and featured work
- `collections.html` - grouped task library with search and preview
- `explorer.html` - live task viewer for interactive browsing

## Structure

- `task-files/` contains a copied snapshot of the original HTML, CSS, JS, and image files
- `styles.css` contains the custom responsive design
- `app.js` renders the grouped task lists, modal previews, and explorer viewer

## GitHub Pages

Deploy the full `project1_portal` folder as a static site repository or place it at the root of an existing GitHub Pages repository.
